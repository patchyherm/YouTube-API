# YouTube Data API – Internal Research Client

**Owner:** Patrick Hermelin · patrick.t.hermelin@gmail.com

This is a **private, single-user** Jupyter workflow using the YouTube Data API to discover and evaluate creators. It accesses **public metadata only** and does **not** redistribute YouTube data/content.

## API usage
- `search.list` (video/channel discovery)
- `channels.list` (channel metadata/statistics)
- `videos.list` (video metadata/statistics)

## Privacy & Compliance
No scraping; API-only. No public UI. API key in env vars; data stored privately (CSV/XLSX). Complies with YouTube API Services Terms and branding.
